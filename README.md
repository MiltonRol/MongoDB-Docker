### Objectius específics de la pràctica
1.  Comprendre i utilitzar Docker Compose per orquestrar serveis.
2.  Configurar volums Docker per garantir la persistència de dades.
3.  Realitzar operacions CRUD sobre una base de dades MongoDB.
4.  Escriure i avaluar consultes amb filtres, projeccions i operadors avançats.
5.  Crear i gestionar índexs per optimitzar el rendiment.
6.  Documentar un projecte tècnic de forma professional.
